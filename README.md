# Benevolence App

The Benevolence App is a solution that integrates with Planning Center to provide meals, services, and item donations to those in need. It was developed in response to the lack of a built-in solution within Planning Center.

In order to use this solution your organization needs a Planning Center account. Don't have one? Sign up for a free here: [https://www.planningcenter.com/](https://www.planningcenter.com/)

## Features

* Integrates with Planning Center People to create users / admins
* Admins and Users that are part of multiple organizations can select which one to sign-in to
* **Users**
  * Login using their email address in PCO People
  * Sign up for requests and get email notifications
* **Admins**
  * Editors and Managers in PCO People are admins in Benevolence App
  * Create requests for Donations, Meals, and Services with a recipient and coordinator
  * View basic analytics in the Admin page
  * Sync with PCO People to update users in Benevolence App

## Getting Started

### Authorize Benevolence App

**NOTE:** Make sure you are logged in to Planning Center with the organization you want to authorize. You must be an Organization administrator, Manager, or Editor in order to authorize Benevolence App.

Go to [Benevolence App](https://app.benevolenceapp.com/login) and select "Authorize with Planning Center".

## Requests

Requests are the bread and butter of Benevolence App. If you're an admin, click on New Request to begin creating one.

Fill out the request form:

* Select a request type (Donation, Meal, or Service) and enter title
* Specify a Recipient and Coordinator (their contact information will be displayed for people to contact)
* Add notes, dates & times
* Address where the request will be fulfilled

### Donations

Donations are requests for meeting tangible needs such as:

* Food and beverages
* Clothing and personal items
* Household supplies
* Financial assistance
* Toys, books, and school supplies

You can specify item names and the quantity needed. People can sign up for some or all of each item.

### Meals

Meals are requests for giving food for events such as:

* Birth of a child
* Loss of loved one
* Church potlucks

**NOTE:** When filling out a Meal request, the delivery dates you select CANNOT be changed after creation. Make sure you verify the dates before submitting the form.

### Services

Services are requests for providing assistance such as:

* Yard work
* Transportation
* Cleaning

## Frequently Asked Questions

### How do I sign-up / cancel an assignment for a request?

Go to the Dashboard and select a request. If there are slots available, click the Actions button and sign up. You will receive an email notification confirming your slot. If you've already signed up and need to cancel, click the Actions button and Unassign. You will be removed and receive an email notification confirming the cancellation.

### How do I see who is providing for a request?

Donations - Click the Actions button and select Details.

Meals - You can see the person's name and what they are providing in the request itself.

Services - Click the Actions button and select Details.

### How do I see basic analytics for my organization?

If you're an admin, go to the Admin page and you will see graphs showing:

* Total number of people synced over
* Total number of requests
* Number of unfulfilled requests (still have unfilled slots)
* Breakdown of requests based on types

### Why am I not seeing updates when I make changes in Planning Center People?

Make sure you sync your organization! Go to the Admin page and select Sync Now. Once complete, you should see those changes reflected in Benevolence App.
